# Mobile Claude Code Confirmation

## Agent Handover Prompt

You are working on the Saroulla/pa-orchestrator repository on the **phase-2-maker-beta** branch.

**Before proceeding with any work:**

Report the current commit SHA of the `phase-2-maker-beta` branch head.

Once reported, the user will verify this SHA against their git app. If it matches, you can safely proceed with file edits and pushes to phase-2-maker-beta.

---

### How to Use

1. Paste this prompt when starting a Claude Code session
2. Wait for the agent to report the commit SHA
3. Verify it matches the current SHA in your git app (GitHub, GitKraken, CLI, etc.)
4. Once confirmed, authorize the agent to proceed with changes
